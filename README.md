Introduction App
By: Sylas Sanchez and Isaiah Urquilla

This is a simple introduction app about the developers who made it
You can run the App by having it hosted using Expo on any device, once hosted you can scan the QR code it provides to view it on moblie or use a link to view it on a browser across any device

Features include: About me for both devs, Gallery, About the app, links to both dev's Githubs, Our Goals and Plans, and more
